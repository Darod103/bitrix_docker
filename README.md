# 1c-Bitrix минимальный набор в Docker для локальной разработки

## ⚙️ Установка 

1. Установите **Docker** и **docker-compose**.

2. Клонируйте репозиторий:

3. Скопируйте .env файл из шаблона:
```bash
cp .env.example .env
```

4. Откройте файл .env и укажите свои данные

5. Запускаем установшик битрикса перейдя в браузере по сылка http://localhost/bitrixsetup.php

5. При созданиее бд не забываем указать заместо localhost устанавливаем db
